    Linux

    一、Linux基础

    1.简介

    • Linux是多用户、多任务、抢占式操作系统，原著者Linus是开源操作系统

    • 基本组成：内核、Shell、System Applications

    • shell提供内核的保护层

      • 对用户屏蔽内核的复杂性
      • 保护内核以免用户操作造成伤害

    • 系统各层次关系：

      • 硬件是实体工作者
      • 内核能够控制硬件工作
      • shell是用户与内核沟通的桥梁，是中间站

    2.基本命令
    • 登录 退出
    • date 查看当前日期
    • cal 查看日历
      • cal 年份 查看当年12个月日历
      • cal 月份 年份 查看该年该月日历
    • clear 清屏
    • echo 输出
    • who 查看当前用户及登陆时间
    • whoami 查看当前用户
    • mail 离线
    • write 在线
    • mesg 信息接收与否的开关命令
    • man+命令 查看具体命令的说明 （press q to quit）

    超级用户root，Shell默认提示符：#

    普通用户，Shell默认提示符：$

    二、文件系统

    文件系统：文件系统实现数据存储的物理（介质）独立性

    文件：是Linux用来存储信息的基本结构，它是被命名（称为文件名）的、存储在某种媒介（如磁盘、光盘和磁带等）上的一组信息的集合。

    1.文件类型
    （1）普通文件（regular file）

    • 二进制文件

    • 文本文件

    （2）目录文件（directory） 包含文件的容器，用于存放目录中文件列表信息
    （3）设备文件（device file） 在linux中，设备被当作文件来处理

    Everything is a file！

    2.目录组织
    • 访问一个文件或子目录时，首先访问他所在的目录，找到该文件的i-node值，然后再查找i-node表找出相应的i-node项，从而找到了相应的数据。

    3.操作命令
    • touch 创建文件
    • cp 复制
    • mv 移动或改名
    • rm 删除文件
    • mkdir 创建文件夹
    • rmdir 删除文件夹
    • whereis 显示文件目录
    • locate 与 find -name 相同
    • ln
      • ln -s abc linkabc （为abc文件创建软链接linkabc，若abc丢失，linkabc会失效）
      • ln abc linkabc（为abc创建硬链接linkabc，abc与linkabc各项属性相同）
    • mount 挂载文件系统
    • ls
      • ls -a 显示全部文件
      • ls 显示文件
      • ls -l显示长目录
      • ls -R 显示文件+目录下文件
    • cd 切换路径
    4.基本概念

    • pwd 当前目录即当前工作目录

    • 用户主目录：/home/rainnn （~）

      cd 回到主目录

    • 绝对路径

    • 相对路径

    • 符号链接（软链接） 可跨文件系统

      • 以路径的形式存在，类似于windows操作系统中的快捷方式
      • 可以对一个不存在的文件名进行链接

    • 硬链接 不可跨文件系统

      • 以文件副本的形式存在，但不占用实际空间
      • 不允许给目录创建硬链接
    5.权限控制

    确定用户对某资源有或无某种操作的权力。

    (1)用户类别

    文件所有者 u

    同组用户 g

    其他用户 o

    所有用户 a

    （2）三种权限

    读 r: 4

    写 w: 2

    执行 x: 1

    -: 0

    （3）权限判断

    ls -lR 查看目录或文件的权限

    • 修改：文件有w权限，文件所在目录有x权限
    • 拷贝：文件有r权限，所在目录有x权限
    （4）权限操作命令
    • chmod
      • chmod a+r 文件名（给所有用户加r权限）
      • chmod u-r 文件名（给文件所有者减r权限）
      • chmod -R a+r *（将目录下所有文件与子目录皆设为任何人可读）
      • chmod 771 file（与chmod ug=rwx,o=x file 效果相同）
    • chown 设置文件所有者和文件关联组
    • su + 用户名 切换用户
    • sudo 以系统管理员的身份执行指令
    6.进程管理
    （1）概念
    • 程序是静态的，是保存在磁盘上的可执行代码和数据的集合
    • 进程（进程是程序执行的过程，是执行的程序）是动态的，是Linux系统的基本调度单位
    • Linux可以同时启动多个进程
    （2）Init进程
    • 内核启动后的第一个进程，其PID=1
    （3）操作命令
    • ps 显示当前进程的状态
    • top 动态地持续监听进程地运行状态
    • kill 删除执行中的程序或工作
    • jobs 可以用来查看当前终端放入后台的工作
    • fg 用于将后台运行的或 挂起 的任务（或作业）切换到前台运行
    • bg 使后台的任务继续运行
    • nohup (no hang up)（不挂起），用于在系统后台不挂断地运行命令，退出终端不会影响程序的运行
    • 前台进程：跟用户交互的
    • 后台进程

    4.变量

    变量名只能是字母、数字、下划线的组合，不能以数字开头

    通常：

    大写字母为系统预设的变量

    小写字母用于用户自定义的变量

    （1）赋值
    • 变量名= 值 或 {变量名= 值}
    abc=1234  或   {abc=1234}
    • 若赋空值：
    abc=""
abc=
abc=''
    • 等号两边不能直接接单独的空格符
    • 如果变量所取的值中包含空格，必须使用引号将变量取值括起来
    name='my name'
name="my name"
name=my\ name
    （2）显示变量
    echo $abc 或  echo ${abc}
    （3）显示所有变量
    set
    (4)清除变量
    unset abc

    清除后变量的值为空

    用于测试一种或几种条件

    1.格式
    • ```shell
      test expression
      
- ```shell
  [ expression ]
    2.输出
    • 条件为真：0
    • 条件为假：1（非0）
    3.操作符
    （1）字符串操作符
    • str1 = str2 字符串相等
    • str1 != str2 字符串不相等
    • str 字符串非空
    （2）整数操作符
    • int1 -eq int2 数值相等
    • int1 -ne int2 数值不等
    • int1 -lt int2 int1<int2
    • int1 -gt int2 int1>int2
    • int1 -le int2 int1<=int2
    • int1 -ge int2 int1>=int2
    （3)文件操作符：测试文件状态
    • -e file 文件file存在
    • -d file 文件file是一个目录
    • -f file 文件file是一个普通文件
    • -s file 文件file大小不为0
    • -r file 可读
    • -w file 可写
    • -x file 可执行
    （4）逻辑操作符：测试多个条件的与或组合
    • -a 逻辑与
    • -o 逻辑或
    • ! 逻辑非

    3.执行时传递参数
    • $0 执行的文件名（包含文件路径）
    • $1 第一个参数 $2 第二个参数 $3 第三个参数 …
    • $# 传递到脚本的参数个数
    • $* 传递的参数作为一个字符串显示
    • $@

    $*和$@区别：

    只有在双引号中体现出来。假设在脚本运行时写了三个参数 1、2、3，，则 “ * “ 等价于 “1 2 3”（传递了一个参数），而 “@” 等价于 “1” “2” “3”（传递了三个参数）。

    传参方法：

    $ bash test 1 2 3

    五、例题

    （一）shell编程

    1.计算两个正整数的最大公约数
    #!/bin/bash
read num1
read num2
while [ $num1 -ne $num2 ]
do 
if [ $num1 -gt $num2 ]
then
num1=$((num1 - num2)) 
else
num2=$((num2 - num1))
fi
done
echo $num1
    2.给出输入的一系列整数的最大数、第二最大数
    #!/bin/bash
read n
max=$n
sMax=$n
while [ $n -ne 0 ]
do
read n
if [ $n -gt $max ]
then
sMax=$max
max=$n
elif [ $n -gt $sMax ]
then
sMax=$n
fi
done
echo $max
echo $sMax
